Nowe możliwości CSS

  1. Zdeprekowane elementy HTML sa powoli usuwane z użycia, a ich funkcje są zastępowane przez odpowiednie style CSS.
  2. Dokumenty pisane z wykorzystaniem CSS są zwykle bardziej prostsze do zrozumienia.
  3. Style pozwalają w łatwy sposób zarządzać duza iloscia stron, poprzez stosowanie zewnętrznych arkuszy stylów
  4. Formatowanie jest o wiele prostsze dzieki skladni CSS
  5. Możliwosc wyboru wielu jednostek oraz kolorow
  6. Dodatkowe możliwości formatowania tekstu
  7. Duza ilosc filtrow takich jak backdrop
