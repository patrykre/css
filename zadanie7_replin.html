1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego,

    2. Styl lokalny pozwala na zastosowanie specyficznego formatowania do konkretnego elementu na stronie internetowej. Element ten będzie formatowany z użyciem stylu lokalnego, co oznacza, że tylko ten konkretny element będzie poddany określonym zmianom wizualnym, a nie wszystkie elementy o tej samej klasie czy tagu.

  2. Składnia teoretyczna ( z użyciem ramki otaczającej ) może być tabela o jednej komórce, zastosuj „znaki specjalne” do napisania definicji stylu lokalnego tak aby instrukcja CSS nie wykonywała się a tylko wyświetlała wewnątrz ramki [  spacja niełamana] przykład „znaku specjalnego”),

    3. lub inaczej

  3. Co może być selektorem,

    4. Selektor w CSS to element, klasa, identyfikator lub atrybut, który określa, do którego elementu HTML lub grupy elementów ma zostać zastosowany styl. Może to być na przykład nazwa tagu HTML, klasa, identyfikator lub atrybut elementu.

  4. Co wipisujemy jako "cechę". Jakie jest znaczenie średników,

    5. W CSS, cechą jest właściwość, która określa wygląd lub zachowanie elementu. Na przykład, kolor, rozmiar czcionki, marginesy, itp. Średniki w CSS służą do oddzielenia deklaracji stylu od siebie. Każda deklaracja stylu składa się z właściwości i wartości, a średnik jest używany do oddzielenia kolejnych deklaracji.

  5. Co wipisujemy jako "wartość" (w deklaracji stylu).

    6. Wartością jest konkretna wartość, która jest przypisywana do właściwości. Na przykład, dla właściwości "color" (kolor), wartością może być nazwa koloru "red", wartość szesnastkowa "#ff0000"/"#f00" lub wartość RGB "rgb(255, 0, 0)".
