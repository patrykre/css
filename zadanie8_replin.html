    2. Styl lokalny jest wstawiany bezpośrednio wewnątrz tagu HTML, który definiuje dany element.

  2. składnia teoretyczna (w ramce, konieczne wcięcia),

  3. jakie mogą być selektory,

    4. Selektory w CSS mogą być różnymi elementami, klasami, identyfikatorami lub atrybutami, które określają, do których elementów HTML ma zostać zastosowany styl.

  4. jakie mogą być cechy i wartości.

    5. Cechą jest właściwość, która określa wygląd lub zachowanie elementu, np. kolor, rozmiar czcionki, marginesy, itp. Wartością jest konkretna wartość, która jest przypisywana do właściwości, np. nazwa koloru, wartość szesnastkowa lub wartość RGB.
